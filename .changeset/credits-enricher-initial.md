Initial release of credits-enricher package

- CreditsEnricher class for augmenting harmonized tracks/releases with detailed credits
- Muso.ai provider with API v4 integration for credits lookup by ISRC
- Credits caching layer with Redis support (7-day TTL by default)
- Matching utilities for ISRC validation and string similarity
- Type definitions for enriched credits with role categorization
